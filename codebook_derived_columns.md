# Derived Column Codebook

| derived_column                     | output_values                              | creation_rule                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
|:-----------------------------------|:-------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| breach_delayed_diagnosis           | YES/NO/UNKNOWN                             | YES if Delayed_Diagnosis_Alleged==YES or Alleged_Breach_Categories contains delayed diagnosis / failure-to-diagnose / misdiagnosis / failure-to-order-imaging terms; NO if allegation field==NO or breach list present without those terms; otherwise UNKNOWN.                                                                                                                                                                                                                                         |
| breach_surgical_technique_error    | YES/NO/UNKNOWN                             | YES if Alleged_Breach_Categories contains surgical-error terms (e.g., negligence in surgical care, foreign object retention, wrong tissue removed, failed appendectomy, untrained intern, postoperative leak) or if Wrong_Structure_Removed / Stump_Leak_or_Stump_Problem / Problematic_Visualization_Alleged / NonSpecialist_Repair_or_Management / Appendix_Not_Removed_or_Wrong_Tissue == YES; NO when relevant source fields/breach list are present without a positive signal; otherwise UNKNOWN. |
| breach_failure_to_remove_appendix  | YES/NO/UNKNOWN                             | YES if Alleged_Breach_Categories contains failure-to-remove-appendix terms or if Appendix_Not_Removed / Appendix_Not_Removed_or_Wrong_Tissue / Wrong_Structure_Removed == YES; NO when source evidence exists and no positive signal is present; otherwise UNKNOWN.                                                                                                                                                                                                                                    |
| breach_improper_postop_management  | YES/NO/UNKNOWN                             | YES if Improper_Postop_Management_Alleged==YES or breach list contains postoperative-management terms (postoperative antibiotics, delayed foreign-body discovery, drain issues, etc.); NO if allegation field==NO or breach list present without those terms; otherwise UNKNOWN.                                                                                                                                                                                                                       |
| breach_failure_to_refer            | YES/NO/UNKNOWN                             | YES if Failure_to_Refer_Alleged==YES or breach list contains failure-to-refer / transfer / timely-surgical-evaluation terms; NO if allegation field==NO or breach list present without those terms; otherwise UNKNOWN.                                                                                                                                                                                                                                                                                 |
| breach_inadequate_informed_consent | YES/NO/UNKNOWN                             | YES if Inadequate_Informed_Consent_Alleged==YES or breach list mentions consent; NO if allegation field==NO or breach list present without consent terms; otherwise UNKNOWN.                                                                                                                                                                                                                                                                                                                           |
| breach_communication_failure       | YES/NO/UNKNOWN                             | YES if Poor_Communication_Alleged==YES or breach list contains communication / notification / misreporting terms; NO if allegation field==NO or breach list present without those terms; otherwise UNKNOWN.                                                                                                                                                                                                                                                                                            |
| perforated_or_gangrenous           | YES/NO/UNKNOWN                             | Combined from Disease_State_at_Presentation and Perforated_or_Gangrenous_Appendix. YES if disease state is perforated appendicitis or gangrenous-necrotic appendicitis, or explicit perforated/gangrenous flag==YES. NO if disease state is uncomplicated/chronic-recurrent or explicit flag==NO. abscess-phlegmon alone is kept as UNKNOWN unless explicit perforation/gangrene is documented.                                                                                                        |
| difficult_case_composite           | YES/NO/UNKNOWN                             | YES if Difficult_Case==YES, Difficulty_Documented is explicit/inferred, Difficulty_Recognized_By_Surgeon==YES, or any explicit difficulty marker (e.g., severe inflammation, adhesions, difficult dissection, conversion, bowel resection, stump problem) == YES. NO only if Difficult_Case==NO or a fully assessable case has explicit negative difficulty documentation. Otherwise UNKNOWN.                                                                                                          |
| plaintiff_favorable                | YES/NO/UNKNOWN                             | Derived from Legal_Outcome and, when available, Appellate_Status. YES for Plaintiff-favorable or Settlement, and for appellate statuses containing plaintiff win. NO for Defense-favorable or appellate statuses containing defense win. Mixed / Unknown / missing remain UNKNOWN.                                                                                                                                                                                                                     |
| plaintiff_age_group                | pediatric / adult / elderly / unknown      | Parsed from Plaintiff_Demographics, Case_Name, and First_Pass_Rationale. Numeric age <18 => pediatric, 18-64 => adult, >=65 => elderly. Otherwise explicit keywords/roles (minor/infant/child; adult; mother/father/doctor/marine/seaman/prisoner/inmate/detainee; World War II veteran) are used when stated.                                                                                                                                                                                         |
| plaintiff_gender                   | Male / Female / unknown                    | Parsed from Plaintiff_Demographics, Case_Name, and First_Pass_Rationale using explicit sex/gender terms only. Mixed-sex multi-plaintiff descriptions or “gender not specified” are coded unknown.                                                                                                                                                                                                                                                                                                      |
| inmate_case                        | YES/NO/UNKNOWN                             | YES if Plaintiff_Demographics, Case_Name, Facility_Type, Defense_Strategy_Summary, or First_Pass_Rationale explicitly mentions inmate/prisoner/incarcerated/detainee/jail/correctional/prison. NO when text is present and none of those indicators appear; UNKNOWN only if no searchable text is available.                                                                                                                                                                                           |
| operative_approach                 | laparoscopic / open / conversion / unclear | Derived from Procedure_Approach, Conversion_to_Open, and explicit operative snippet terms. Conversion_to_Open==YES overrides. laparoscopic includes robotic/minimally invasive descriptions. open includes explicit open/laparotomy wording. Otherwise unclear.                                                                                                                                                                                                                                        |
| adaptation_performed               | YES/NO/UNKNOWN                             | YES if Adaptation_Performed==YES or Adaptation_Type is a specific non-none value (e.g., drain/other). NO if Adaptation_Performed==NO or Adaptation_Type==none. Otherwise UNKNOWN.                                                                                                                                                                                                                                                                                                                      |
| death_or_long_term_morbidity       | YES/NO/UNKNOWN                             | YES if Death==YES or Long_Term_Morbidity==YES. NO only if both Death==NO and Long_Term_Morbidity==NO. Otherwise UNKNOWN.                                                                                                                                                                                                                                                                                                                                                                               |
| high_severity_injury               | YES/NO/UNKNOWN                             | YES if Injury_Severity==major or if Death / Long_Term_Morbidity / Need_for_Bowel_Resection / Need_for_Stoma == YES. NO if Injury_Severity==minor and none of those severe-outcome flags are YES. Otherwise UNKNOWN.                                                                                                                                                                                                                                                                                    |
